B. Description
1. Abstract  Polygons defining county-level political or administrative units (MQ units, for which urban qu are merged into a single polygon). Region subclasses define mergers of county-level units for analytical purposes (MC units), for cross-time comparisons (XT units), or provinces (PR units).
2. Purpose Spatial framework for managing and analyzing tabular data compiled for county-level units at 1:1,000,000 scale.
3. Procedures Used (overview) Boundaries provided by ACASIAN and modified based on other paper map sources (noted in Arc SOURCE item). INFO tables define which polygons are used to build each MQ, MC, or XT unit; when changes are made to these definitions, region subclasses are reconstructed by REBUILDCHINA-A.AML.  See MC Merger Procedures.
4. Sources and related Data Sets County boundaries were originally georeferenced by ACASIAN, Griffiths University, Australia (q.v.) from the Land Use Map of China and other sources. 

County-level identification numbers from China-Z and China-A Datasets (for 1982-85 and 1990, respectively), University of California, Davis.
