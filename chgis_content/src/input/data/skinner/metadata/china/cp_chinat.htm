CP/China-T

I. Identification Information

A. Citation

1. Title Central Places of China (provisional), 1990
2. Version / Date August 1996
3. Originator G.W. Skinner Research Team, Department of Anthropology, University of California, Davis
4. Date and Place of Publication Davis, California: 1998

B. Description

1. Abstract Points indicating the location of some major central places (cities, towns, and villages) which are included in the China-T. Includes all greater cities and some smaller central places.
2. Purpose Spatial framework for managing and analyzing tabular data compiled for central places at 1:1,000,000 scale.
3. Procedures Used (overview) The China-T coverage was used for cartographic purposes prior to the acquisition of the CPxx series of coverages, which include more, smaller central places. Most analyses should now use CPxx or coverages derived from that series.
4. Sources and related Data Sets China-T identification numbers and data values derived from the China-T Dataset, University of California, Davis.

II. Data Quality Information

1. Intended area of coverage Mainland China
2. Completeness Includes points for all greater cities (and higher levels of the central place hierarchy), and selected smaller cities.
3. Processing steps See .log file.

1. Attribute Label 2. Attribute Description 3. Attribute Domain or Table

B. Point Attribute Table (PAT)

NAME Place name in Pinyin Derived from China-T Dataset's T3 field but shortened for cartographic purposes.
NAMECAP Place name in Pinyin, capitalized NAME in all capital letters (populated for selected major cities only)
T1 China-T Identifier 8-digit identifier used in China-T Dataset.
Other attributes in the .PAT are based on outdated versions of the China-T Dataset, or are undocumented. These attributes should not be used for future analysis.
