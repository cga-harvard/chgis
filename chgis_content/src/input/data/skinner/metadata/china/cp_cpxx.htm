B. Description
1. Abstract  Points indicating the location of central places (cities, towns, and villages) shown on the Land Use Map of China and other published sources; includes most of the 12,000+ places included in the China-T Dataset. This information is published in a series of Arc/Info coverages, with one coverage for each of China's provinces, autonomous regions, and independent municipalities. Coverages are named CPxx, where xx is the two-digit identifer for each province.
2. Purpose Spatial framework for managing and analyzing tabular data compiled for central places at 1:1,000,000 scale.
3. Procedures Used (overview) ACASIAN digitized points from paper map sources and compiled data on each point. UCD received this series from ACASIAN and processed this data for use with the China-T Dataset, assigning each point a T1 identification number and verifying certain place names. Some places listed in the China-T Dataset, but not included in this series, are included in a supplemental coverage (CPQU).
4. Sources and related Data Sets Source maps used by ACASIAN may include the Land Use Map of China and other publications. 

Urban population and China-T identification numbers derived from the China-T Dataset, University of California, Davis.

II. Data Quality Information

1. Intended area of coverage Mainland China (separate Arc/Info coverages for each province)
2. Completeness Includes points for over 12,000 cities, towns, and villages in all provinces of China. Some places listed in the China-T Dataset, but not included in this series, are included in a supplemental coverage (CPQU).
3. Processing steps See .log files and the CP-.AML series for details.

ACASIAN digitized points from paper map sources and compiled data on each point, including its name (in hanzi [Chinese characters] and pinyin [transliterated into Latin characters]), administrative identification number (CITY_TOWN_CODE), official population, and position in the administrative hierarchy. This product is distributed for a fee exclusively by ACASIAN.

UCD received this series from ACASIAN in March 1997 and processed this data for use with the China-T Dataset (tabular data on central places compiled from China's 1990 Census and other published sources). 

Supplemental data attributes compiled by ACASIAN were moved from the .PAT file to a separate .CTC file keyed to the points' unique identifier, the CITY_TOWN_CODE. These attributes were not verified or edited by UCD. The CITY_TOWN_CODE was also retained in the .PAT as a unique identifier for use in retrieving data in the .CTC file or other data at ACASIAN.

UCD changed the pinyin names listed in the .PAT where the China-T Dataset used different names for the same cities; we also added hanzi names for certain higher-order cities whose hanzi names had been omitted from ACASIAN's product. We also removed the shi, xian, and zhen suffixes from both hanzi and pinyin names except when the remaining portion of the names would have only one syllable.

Next, UCD matched each point to a corresponding record in the China-T Dataset, using as guides the CITY_TOWN_CODE, the pinyin name, the name and ID number of the county in which the point is located, and the approximate population of the point as shown in both the .CTC file and the China-T Dataset. Each point was assigned a T1 identification number corresponding to a record in the China-T Dataset. In a majority of cases, the T1 identifier was the same as the CITY_TOWN_CODE. However, further processing was required to resolve over one thousand discrepancies.

This series included points some minor places which are not listed in the China-T Dataset. Likewise, some central places listed in the China-T Dataset were not included in this series; these included all urban qu centers. A supplemental coverage (CPQU, q.v.) was created to contain many of these points. All additions, corrections, and unresolved discrepancies noted at UCD are being provided to ACASIAN to assist in the further development of their commercial products.

B. Point Attribute Table (PAT)
CHARACTER_NAME Place name in Chinese characters (EUC encoding) Hanzi (Chinese character) name.

The 100 most frequently used characters are translated in ChinesePlaceNames.htm.

Printing these Chinese character names in Arc/Info is possible with the installation of the East Asian Languages Support CD, available from ESRI. Hanzi names can be displayed or edited in a Telnet session from a PC which is also running TwinBridge software.
CITY_TOWN_CODE ACASIAN Identifier This item is retained for consistency with the data attributes originally provided by ACASIAN, which have now been moved to the .CTC file.
NAME Place name in Pinyin Derived from ACASIAN's PINYIN_NAME attribute and verified against China-T Dataset's T3 field.
T1 China-T Identifier  8-digit identifier used in China-T Dataset. Based on China's National Standard (guobiao) codes for early 1990, with significant changes made at UCD for analytical purposes. 
T1_C China-T Identifier (redefined as Character data type) A redefined INFO item. Used when it is necessary to treat T1 as a character data type, such as when comparing with CITY_TOWN_CODE. 
T1_6 China-T Identifier (redefined as Character, excluding first two digits) A redefined INFO item: T1 without the first two digits (which indicate the province). Used on maps where the province number can be assumed. Defined as Character data type so that leading zeroes will be printed.
C. .CTC Table (Attributes provided by ACASIAN)

These attributes will not be verified or edited by UCD. Tables for certain provinces contain additional attributes, often containing "old" values for standard attributes.
CITY_TOWN_CODE ACASIAN Identifier 8-digit identifier used by ACASIAN. Based closely on guobiao codes for late 1990.
CHARACTER_NAME Place name in Chinese characters (EUC encoding) Hanzi place name, including shi, xian, or zhen suffix.
PINYIN_NAME Transliterated place name Pinyin place name, including shi, xian, or zhen suffix.
SIX_DIGIT_POSTAL Postal Code 6-digit identifier used for postal delivery.
COUNTY_LEVEL_CAP Capital of a county-level unit YES | NO
OFFICIAL_STATUS Status in late 1990 CITY | TOWN | etc.
CENSUS90_ORDER Numeric code Undocumented
URBAN_RESIDENT_P Population classified as urban (nonagricultural) Comparable to T109 in the China-T Dataset.
Additional Administrative Position Codes --  Identification numbers, hanzi names, and pinyin names of the administrative unit, county, and province to which each place is assigned.
