How to Work in Arcedit like a Pro

but not necessarily an Arcedit Pro

Eight steps for adding Qu Center points

Step What to Type Explanation
1. Start Arc/Info
username
password
arc
Log into a VisLab computer with the project's username and password; start Arc/Info.
2. Set up for editing Qu points 
Arc: &R EDITQU

Enter the 2-digit number of the province you want to work with.

 "&R" means "run an AML script program. The EDITQU program sets things up so that you can edit the coverage (dataset) of Qu points.
3. Add a point In the map window, click the left mouse button on the map at the point you want to add

Type in the Qu's Name and Number in the text window

Confirm that the Name and Number are correct

 Locate the new Qu point relative to the other features on the screen: other central places, red roads and railroads, blue rivers and lakes, green county and provincial boundaries.
Special keys to help when editing:
  • CTRL-W creates a new window; use the mouse to draw a rectangle over the area to be shown in the new window.
  • CTRL-R refreshes the current window
  • CTRL-D gets rid of a zoom window
 Note the X and Y coordinates at the top of the map screen, which can be used to estimate distances.

All windows can be resized at any time; if you make a window bigger, it will show the same map area with more detail.

4. Stop adding points Type 'N' when asked if you want to continue Do this if you want to move to another province, or if you want to delete a point you've added. You will then see the Arcedit prompt.
5. Delete a point you added 
Arcedit: SEL *
Arcedit: DELETE
Click once on the point that you want to delete.
6. Save your work 
Arcedit: SAVE
Do this often!
7. Add more points 
Arcedit: &R ADDQU
Resumes the process of adding points (see step 3)
8. End work 
Arcedit: SAVE
Arcedit: QUIT
Arc: QUIT

Log out of the VisLab computer

 Stop adding points (see step 4), then save your work and quit out of Arcedit and Arc/Info. Finally, log out of the VisLab computer.
