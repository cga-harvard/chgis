Vis Lab workstations

Individual accounts
login: password:

Staff and student workers are provided with Vis Lab accounts with a disk quota of 125mb. All work in progress should be completed within individual accounts or common areas before being transferred to the FZSKIN account.

The FZSKIN account
login: fzskin password:

This account has a disk quota of 750mb and is the repository for completed GIS data sets. Subdirectories include china, france, nobi (Japan), and atool (AML). This directory should be archived to tape once per month.

Common areas

The directory /scratch/tmp/fzskin is available for temporary storage and processing of GIS data sets. As this area is not archived by Vis Lab staff, and files stored there are not protected from access by other Vis Lab users, this directory should not be used for long term storage. After processing, files should be copied back to individual accounts or FZSKIN subdirectories.

Directories named /tmp exist on all Vis Lab workstations, but vary in size from a few megabytes on older workstations to 4 gigabytes on escher. If space is available, these directories may be used for processing GIS data sets; disk access is significantly faster compared with other directories. However, data stored in one workstation's /tmp is not accessible from other workstations, and in all cases the contents of /tmp will be erased whenever a workstation is rebooted.

Vis Lab output devices

There are three output devices in the Vis Lab:

Apple Laserwriter (lpr10) -- black & white Postscript printer

Tektronix Phaser (lpr -Pcolor) -- antiquated color Postscript printer (cost ~$12/page)

HP650c Plotter (plotmap) -- E-size color Postscript plotter (cost ~$50/sheet)

Supplies for these devices are available from the IT consultants. The configuration of the HP650c plotter, in particular, should be checked before use as it may be reset by other users.
