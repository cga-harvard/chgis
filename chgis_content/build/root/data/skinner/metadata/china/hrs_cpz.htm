4. Date and Place of Publication Davis, California: 1999

B. Description
1. Abstract  Polygons illustrating core-periphery zones.
2. Purpose Spatial framework for macroregional analysis at 1:1,000,000 scale.
3. Procedures Used (overview) Core-periphery were delineated on paper maps printed at 1:1,000,000 scale. Zone boundaries were delineated based on core-periphery index values computed for each county-level (MC) unit and on transportation and physiographic features. Boundaries traced on acetate were scanned, vectorized, and edited to form a single coverage.
4. Sources and related Data Sets County boundaries and transportation network georeferenced by ACASIAN, Griffiths University, Australia (q.v.). 

County-level index values from China-A Dataset, University of California, Davis.

A. Arc Attribute Table (AAT)
BTYPE Boundary Type 2 = Arbitrary Boundary under water

3 = Core-periphery zone (CPZ) boundary

4 = CPZ and Macroregional Subsystem Boundary

5 = CPZ and Macroregional System Boundary

B. Polygon Attribute Table (PAT)
MR Macroregion 1..9
PANZONE All-China core-periphery zone  numbering system 1..14
Macroregion-specific core-periphery zones (RxCPZ values) have been interdigitated here to facilitate panregional comparisons.
RxCPZ Macroregion-specific core-periphery zones Nine columns (one for each macroregion).
Valid values range from 1 (Inner Core) to 7 (Far Periphery).  Note that values in each region are not comparable with values in other regions:  Zone 1 in Region 6 (Shanghai) represents quite different conditions than zone 1 in Region 9.
Values of zero indicate polygons not in the specified macroregion.
