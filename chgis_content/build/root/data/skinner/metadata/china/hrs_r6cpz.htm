B. Description

1. Abstract Polygons illustrating core-periphery zones.
2. Purpose Spatial framework for macroregional analysis at 1:1,000,000 scale.
3. Procedures Used (overview) Core-periphery were delineated on paper maps printed at 1:1,200,000 scale. Zone boundaries were delineated based on core-periphery index values computed for each county-level (MC) unit and on transportation and physiographic features. Boundaries traced on acetate were scanned, vectorized, and edited to form a single coverage.
4. Sources and related Data Sets County boundaries and transportation network georeferenced by ACASIAN, Griffiths University, Australia (q.v.).

County-level index values from China-A Dataset, University of California, Davis.
